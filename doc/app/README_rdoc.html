    Installation¶ ↑

    On Debian 8 (Jessie) or later and Ubuntu 14.10 (Utopic Unicorn) or later:

    # apt-get install bundler git nodejs postgresql postgresql-server-dev-9.4 ruby ruby-dev

    On Ubuntu 14.04 (Trusty Tahr):

    # apt-get install bundler git nodejs postgresql postgresql-server-dev-9.3 ruby2.0 ruby2.0-dev

    # rm /usr/bin/ruby /usr/bin/gem /usr/bin/irb /usr/bin/rdoc /usr/bin/erb

    # for f in ruby gem irb rdoc erb; do ln -s /usr/bin/${f}2.0 /usr/bin/$f; done

    # gem update --system

    # gem pristine --all

    Then, on all systems:

    $ git clone https://[username]@github.com/valegro/subsright.git

    $ cd subsright

    $ git checkout andrew

    $ bundle install

    $ sudo gem pristine --all

    $ sudo -u postgres createuser -d $USER

    $ rake db:create

    $ rake db:migrate

    $ rake test

    $ rails s

    You can now visit localhost:3000 which will initialise the default configuration.

    Check log/development.log for the text of the outbound mail to admin@example.com and follow the “confirm your account email” link provided. You can then log in with that email and the initial default password “password”. You can now create and edit admin users and the site configuration.
